# ANTIGRAVITY.md
## 瞬間英作文アプリ（CEFR B2レベル）

### 1. プロジェクトの目的
自分が毎日の学習として手軽にアウトプット（英作文）を鍛えるためのアプリケーション。
PC・スマホのどちらからでもサクッと問題にアクセスし、CEFR B2レベルに特化した英作文トレーニングができる状態を目指す。

### 2. 現在のMVP（最小構成）
- **核となる体験**:
  1. 画面に日本語のお題を出題（まずは動作確認用に5問程度固定で内蔵）。
  2. ユーザーが英語を入力し、[判定] ボタンを押す。
  3. 正解（模範解答）と自分の入力を画面に並べて表示し、簡単な正誤判定を行う。
  4. [次の問題へ] ボタンでループする。
- **デザイン**: PC / スマホ対応のレスポンシブデザイン（Vanilla CSSによるモダンで使いやすいUI）。

### 3. 今後の展望（Phase 2以降）
- **高度なAI添削**: 外部生成AI（OpenAI APIやGemini等）と連携し、模範解答との差分だけではなく、ユーザーが入力した英文の自然さや文法的なミスを添削する機能。
- **データ永続化**: Supabase等のBaaSを利用し、ユーザーの「解答履歴」「正誤データ」を保存できるようにする。この際、RLS（行レベルセキュリティ）による安全な権限管理を行う。
- **エビングハウスの忘却曲線**: 保存された解答履歴をもとに、過去に間違えた問題を最適な復習タイミングで再出題するロジック（spaced-repetitionアルゴリズム）の実装。

### 4. 技術スタック
- **フロントエンド**: React (最新Vite環境)
- **言語**: JavaScript
- **スタイリング**: Vanilla CSS（Tailwind等は不使用）
- **バックエンド（予定）**: Supabase（データベース・認証）
