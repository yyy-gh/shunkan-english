# ANTIGRAVITY.md
## 瞬間英作文アプリ（CEFR B2レベル）

### 1. プロジェクトの目的
自分が毎日の学習として手軽にアウトプット（英作文）を鍛えるためのアプリケーション。
PC・スマホのどちらからでもサクッと問題にアクセスし、CEFR B2レベルに特化した英作文トレーニングができる状態を目指す。

### 2. 現在のMVP（最小構成）
- **核となる体験**:
  1. 画面に日本語のお題を出題（まずは動作確認用に5問程度固定で内蔵）。
  2. ユーザーが英語を入力し、[判定] ボタンを押す。
  3. 正解（模範解答）と自分の入力を画面に並べて表示し、簡単な正誤判定を行う。
  4. [次の問題へ] ボタンでループする。
- **デザイン**: PC / スマホ対応のレスポンシブデザイン（Vanilla CSSによるモダンで使いやすいUI）。

### 3. 現在のステップ（Phase 2: 認証・データ保存）
- **Google OAuth認証**: ユーザー登録のハードルを下げるため、Supabase Authを利用したGoogleログインを採用。
- **データベース永続化**: SupabaseのPostgreSQLを利用し、各ユーザーの「学習履歴（問題文、入力解答、正誤、AIのフィードバック等）」を保存。
- **セキュリティ（RLS）**: データベースにはRLS（Row Level Security）を適用し、ユーザー自身のみが自分のデータを読み書きできる安全な状態を構築する。

### 4. 今後の展望（Phase 3以降）
- **エビングハウスの忘却曲線**: 保存された解答履歴をもとに、過去に間違えた問題を最適な復習タイミングで再出題するロジック（spaced-repetitionアルゴリズム）の実装。
- **UI/UXのブラッシュアップ**: 履歴を振り返れるマイページ画面の作成など。

### 5. 技術スタック
- **フロントエンド**: React (最新Vite環境)
- **言語**: JavaScript
- **スタイリング**: Vanilla CSS（Tailwind等は不使用）
- **バックエンド/DB**: Supabase（Google OAuth認証、PostgreSQLデータベース）
